Add `interactive-panel` for hover lift, cursor sheen, and viewfinder brackets.
Variants: `.glass-panel--navbar-dense`, `.glass-panel--reading`.

## Chamfer utilities

<Demo>
  <div class="glass-panel chamfer-tr" style="width:200px">
    <div class="glass-panel-content pad"><span class="eyebrow">.chamfer-tr</span></div>
  </div>
  <div class="glass-panel chamfer-br" style="width:200px">
    <div class="glass-panel-content pad"><span class="eyebrow">.chamfer-br</span></div>
  </div>
</Demo>

`.chamfer-tr` (flags) and `.chamfer-br` (CTAs) cut a single corner via
`clip-path`. Pair with a host/inset pseudo pair for a hairline border.
